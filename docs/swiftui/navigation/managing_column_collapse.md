# Managing column collapse

## `NavigationSplitViewColumn`

表示导航拆分视图中的列的视图。

在某些情况下，`NavigationSplitView` 会折叠成单个堆栈，例如在 iPhone 或 Apple Watch 上。 将此类型与 `preferredCompactColumn`  参数结合使用可控制导航拆分视图的哪一列显示在折叠堆栈的顶部。
